# KO掌机助手 下载站

KO掌机助手的 GitHub Pages 下载页面。

## 部署到 GitHub Pages

1. 创建新仓库 `paneldownload`
2. 推送本项目到仓库
3. 进入 Settings → Pages
4. Source 选择 `Deploy from a branch`
5. Branch 选择 `main`，文件夹选择 `/ (root)`
6. 点击 Save

## 访问地址

`https://你的用户名.github.io/paneldownload/`

## 文件列表

```
paneldownload/
├── index.html                        # 下载页面
├── KO掌机助手_1.0.0_x64-setup.exe    # 安装版
├── KO掌机助手_1.0.0_绿色版.zip       # 绿色版
└── README.md
```

## 注意事项

- GitHub 单文件限制 100MB，当前文件均在限制内
- 如需更新版本，替换对应文件并修改 index.html 中的版本号
