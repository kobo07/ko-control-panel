# KO掌机助手

专为 Windows 掌机打造的性能调节工具，支持 GPD、AYANEO、OneXPlayer、ROG Ally 等设备。

全界面支持手柄导航，无需键鼠即可完成所有操作。基于 Tauri 构建，轻量高效。

## 功能

**性能调节**
- TDP 功率控制（滑条/预设）
- CPU 频率限制、核心数控制
- GPU 时钟调节
- Turbo Boost 开关

**风扇控制**
- 静音/平衡/性能/狂飙模式
- 自定义温度曲线

**显示设置**
- 分辨率、刷新率切换
- 亮度调节

**输入增强**
- 陀螺仪映射为右摇杆（体感瞄准）
- 快捷键触发

**游戏管理**
- Steam 游戏库扫描
- 自动 TDP 配置
- FLiNG 修改器下载

**GPU 设置 (AMD)**
- RSR 超分辨率
- Anti-Lag、Chill、FreeSync

**系统优化**
- 一键禁用遥测
- 网络快捷开关
- 电源方案导入导出

## 使用

1. 运行程序
2. 手柄按 `LB+RB+Back` 呼出面板（可自定义）
3. 方向键导航，A 确认，B 返回，LB/RB 切换页面

陀螺仪功能需安装 ViGEmBus 驱动（程序内可一键安装）。

## 支持设备

- GPD WIN 系列
- AYANEO 系列
- OneXPlayer 系列
- ROG Ally / Legion Go
- Steam Deck (Windows)
- 其他 x86 Windows 掌机
